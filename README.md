# Mujeres en Videojuegos Competitivos en Latinoamérica

Proyecto final de la clase de **Storytelling**, desarrollado como una experiencia narrativa interactiva enfocada en los desafíos, dinámicas y experiencias de las mujeres dentro de los videojuegos competitivos en línea en Latinoamérica.

El proyecto busca visibilizar diferentes problemáticas, percepciones y dinámicas sociales presentes en comunidades de videojuegos competitivos, mediante una narrativa digital inmersiva basada en scroll storytelling.

## Sobre el proyecto

Este proyecto presenta un recorrido visual e interactivo sobre temas como:

- Participación femenina en videojuegos competitivos.
- Acoso y discriminación en espacios de gaming online.
- Estrategias de adaptación utilizadas por las jugadoras.
- Redes de apoyo dentro de las comunidades.
- Factores sociales que afectan la permanencia y experiencia de las mujeres en entornos competitivos.

La experiencia fue diseñada utilizando un enfoque de **storytelling interactivo**, combinando narrativa, diseño visual y exploración progresiva del contenido para generar una experiencia más inmersiva.

## Cómo ejecutar el proyecto

1. Clona este repositorio:

```bash

```

2. Entra al directorio del proyecto:

```bash
cd mujeres-gaming-latam
```

3. Abre el archivo `index.html` en tu navegador o utiliza **Live Server** en VS Code.
